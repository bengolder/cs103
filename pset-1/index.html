Notes on Proofs

Proofs begin with definitions and assumptions. The statement we are trying to prove is the theorem.

Example proof

Theorem: If $n$ is an integer, then $n^2$ is even.

  • Let $n$ be an even integer.
  • Since $n$ is even, there is some integer $k$ such that $n = 2k$.
  • This means that \( n^2 = (2k)^2 = 4k^2 = 2(2k^2) \)
  • From this, we see that there is an integer $m$ (namely, $2k^2$) where $n^2 = 2m$.
  • Therefore, $n^2$ is even. ■

Checkpoint Problem Two: Multiples of Three

For every integer n, n is a multiple of three if and only if n2 is a multiple of three.

Prove the following statements:

  • For any integer n, if n is a multiple of three, then n2 is a multiple of three.
  • For any integer n, if n2 is a multiple of three, then n is a multiple of three.

Prove the first with a direct proof. Prove the second using proof by contrapositive. Make sure you state the contrapositive explicitly before proving it. Prove, by contradiction, that \( \sqrt 3 \) is irrational.
